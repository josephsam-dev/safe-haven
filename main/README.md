# Safe Haven

## Description
Safe Haven is a web application designed to provide a supportive digital space
where users can share stories, request assistance, and access resources in a
safe and structured environment.

## Features
- User authentication
- Story sharing
- Request management
- Admin/staff tools

## Tech Stack
- Python
- Django
- HTML/CSS
- SQLite/PostgreSQL

## Setup Instructions
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies
4. Run migrations
5. Start the development server
